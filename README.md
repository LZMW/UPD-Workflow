# Unified Development Workflow v1.3

## 简介 / Introduction

一套基于6A框架的统一项目开发工作流规范，专为AI辅助开发设计。

A unified project development workflow specification based on the 6A framework, designed for AI-assisted development.

## 6A框架 / 6A Framework

1. **Align (对齐)** - 模糊需求 -> 精确规范 / Vague requirements -> Precise specifications
2. **Architect (架构)** - 共识 -> 架构设计 / Consensus -> Architecture design
3. **Atomize (原子化)** - 设计 -> 任务拆分 / Design -> Task breakdown
4. **Approve (审批)** - 审查与确认 / Review and confirmation
5. **Automate (自动化执行)** - 按节点执行 -> 测试 -> 代码 -> 文档 / Execute by node -> Test -> Code -> Document
6. **Assess (评估)** - 结果评估 -> 交付 / Result evaluation -> Delivery

## 核心特性 / Core Features

- **规范驱动** / Specification-driven development process
- **质量优先** / Quality-first principles (no delays, no overruns, no errors)
- **深度思考机制** / Deep thinking and tool collaboration mechanisms
- **完整文档体系** / Complete documentation template system

## 质量原则 / Quality Principles

1. **不延期** / No Delays - 严控风险，遇阻即调 / Strict risk control, adjust when blocked
2. **不超支** / No Overruns - 范围/资源严格匹配 / Strict scope/resource matching
3. **不出错** / No Errors - 多轮自检，出错即停 / Multi-round self-checking, stop on errors
